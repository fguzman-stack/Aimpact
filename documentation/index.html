Documentación del Template

Todo lo necesario para instalar, personalizar y desplegar AImpact.

Estructura del proyecto

/
├─ assets/
│  ├─ css/           CSS compilado (style.css)
│  ├─ js/            JS de UI (main.js)
│  ├─ scss/          Fuentes SCSS (variables, mixins, componentes)
│  └─ images/        Imágenes (coloca las tuyas aquí)
├─ pages/            Páginas HTML listas (index, pricing, blog, etc.)
└─ documentation/    Esta guía

Requisitos

  • Navegador moderno (Chrome/Firefox/Edge/Safari).
  • Opcional: Node.js 18+ si deseas compilar SCSS localmente.

Vista local

  1. Abre el directorio raíz del proyecto.
  2. Usa cualquier servidor estático (por ejemplo, http-server o la extensión Live Server) y navega a /pages/index.html.

Compilar SCSS (opcional)

El CSS ya está compilado en assets/css/style.css. Si deseas editar SCSS:

# Instala Sass (una vez)
npm i -D sass

# Compila SCSS a CSS
npx sass assets/scss/style.scss assets/css/style.css --style=compressed --no-source-map

Personalización

  • Colores y tema: variables en assets/scss/_variables.scss y soporte claro/oscuro con data-theme.
  • Componentes y estilos: assets/scss/_components.scss.
  • JS de UI (modo oscuro, navbar sticky, pricing toggle): assets/js/main.js.

Despliegue

  • Sube el proyecto a cualquier hosting estático (Netlify, Vercel, GitHub Pages).
  • Asegúrate de que el documento de inicio sea pages/index.html.

Soporte y licencias

Incluye un archivo README, nota de soporte básico y licencia del comprador. En marketplaces la licencia la provee la plataforma.
