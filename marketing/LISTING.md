Title: AImpact — AI Startup & SaaS HTML Landing Template

Live Preview: https://fguzman-stack.github.io/Aimpact/pages/index.html

Item Description:
AImpact es una plantilla HTML moderna y multipágina diseñada para startups de IA y productos SaaS. Incluye modo oscuro integrado, animaciones AOS, precios con alternancia mensual/anual, componentes UI pulidos y una estructura clara para features, blog, pricing y más. Es ligera, 100% estática y fácil de personalizar mediante variables CSS/SCSS.

Main Features:
- Multipágina con secciones clave (Landing, Pricing, Blog, About, Contact, Dark Mode demo)
- Modo claro/oscuro con persistencia (localStorage) y `data-theme`
- Animaciones AOS y navbar sticky
- Toggle de precios mensual/anual sin dependencias extra
- Variables CSS/SCSS y paletas de color configurables
- Componentes y tarjetas con estados hover y sombras sutiles
- Responsiva y lista para hosting estático (Netlify/Vercel/GitHub Pages)
- Documentación incluida

Pages Included:
- Landing: pages/index.html
- Dark Mode: pages/index-dark.html
- Pricing: pages/pricing.html
- Blog (lista y post): pages/blog/index.html, pages/blog/single.html
- About/Contact/FAQ/Features/Portfolio/Team/404

Technical:
- HTML5 + Bootstrap 5.3 (CDN)
- Animations: AOS (CDN)
- Iconos: Font Awesome (CDN)
- CSS listo (`assets/css/style.css`, `style.min.css`) y SCSS fuente (`assets/scss/*`)
- JS ligero para UI (`assets/js/main.js`)
